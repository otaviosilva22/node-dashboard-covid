Brasil.io

Todos os dados utilizados na aplicação são captados a partir da API disponibilizada pela plataforma Brasil.io.

A plataforma conta com 40 voluntários, que diariamente contribuem na compilação de boletins epidermiológicos das 27 secretarias Estaduais de Saúde. Deste modo, é possível acessar informações sobre a COVID-19 por município, são elas:
- Total de casos
- Total de mortes
- Novos casos
- População contaminada

Neste sentido, são disponibilizadas também outras bases estruturadas, como: população dos municípios afetados, óbitos suspeitos e etc.
