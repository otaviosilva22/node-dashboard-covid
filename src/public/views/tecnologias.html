Sobre

Para o front-end do site utilizamos HTML5, CSS, Bootstrap e a biblioteca Chart.js para geração dos gráficos responsivos.

Já o back-end, por sua vez, foi desenvolvido todo em Node.js com auxílio do framework Express para gestão de requisições e respostas das APIs.

A comunicação do front-end com o back-end foi feita a partir do jQuery, mais especificamente com a função Ajax.
